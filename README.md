## Que hacemos en Kpixel?

Somos un emprendimiento de desarrollo de software enfocado en construir soluciones digitales que realmente funcionen para tu negocio.

**Paginas web y landing pages:** rapidas, modernas y optimizadas para cualquier dispositivo.

**Plataformas web a medida:** sistemas de gestion, dashboards y portales empresariales.

**Automatizacion de procesos:** ahorra tiempo y reduce errores manuales en tu operacion.

**Aplicaciones web:** frontend y backend robustos, desde cero o sobre sistemas existentes.
